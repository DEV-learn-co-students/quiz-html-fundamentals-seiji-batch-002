## Quiz 2 - HTML Fundamentals

Make a note of your selected answers for the questions below. The correct answers are provided at the end of the quiz along with an explanation.

1. Describe the purposes of the &lt;!doctype&gt; element.  
A.) Refers the browser to another page upon loading.  
B.) Informs the browser of the documents HTML version.  
C.) Signals a comment in the code.  
D.) None of the above, this is not a valid HTML element.

2. Describe the purposes of the &lt;html&gt; element.  
A.) Tells the browser to ask the server for another HTML page.  
B.) Comments out the HTML code in between.  
C.) Defines the starting point and ending points of the HTML document.  
D.) None of the above.

3. Describe the purposes of the &lt;head&gt; element.  
A.) Defines the section of the HTML document that contains the title, meta data, and other browser related script, css, and shortcut icon links.  
B.) Tells the browser that the start of the document is coming.  
C.) Used to define the heading bar that appears at the top of page layouts usually containing main page navigation.  
D.) None of the above.

4. Describe the purposes of the &lt;body&gt; element.  
A.) Defines the invisible parts of the page for search engines.  
B.) Tells the browser to remove spacing around the elements within.  
C.) Tells the browser to add scrollbars to the content below.  
D.) Defines the section of the HTML document that contains all of the visible page content such as images, text, links, and page layout.

5. Describe the purposes of the &lt;title&gt; element.  
A.) Defines the part of the page with the largest heading over 18px.  
B.) Defines the part of the page where the main navigation will sit.  
C.) Gives a title to the website that will be used to style all pages under that style.  
D.) Defines the pages main title. This appears at the top of the browser tab. It is important to search engines.

6. Describe the purposes of the &lt;h1&gt; element.  
A.) Contains useful key terms for search engines.  
B.) Defines a level 1 heading.  
C.) Suggests that the content is hot.  
D.) Both A and B.

7. Describe the purposes of the &lt;p&gt; element.  
A.) Contains readable text content.  
B.) Defines a paragraph of text.  
C.) Allows spatial relationships between the paragraph and other elements above and below.  
D.) All of the above.

8. Describe the purposes of the &lt;ul&gt; element.  
A.) Defines an unordered list of items.  
B.) Defines to underline the content.  
C.) Asks the browser to highlight the code below.  
D.) None of the above.

9. Describe the purposes of the &lt;ol&gt; element.  
A.) Tells the browser to overline the text inside.  
B.) Adds an outline around the element.  
C.) Defines an ordered list of items.  
D.) None of the above.

10. Describe the purposes of the &lt;li&gt; element.  
A.) Defines a listen element that plays a sound.  
B.) Creates list items to exist inside either &lt;ol&gt; or &lt;ul&gt; parent elements.  
C.) Tells the browser to italicise the text.  
D.) None of the above.

11. Describe the purpose of the &lt;img&gt; element.  
A.) Defines an image to be loaded inside the document.  
B.) Defines an imaginary line inside the document used for planning.  
C.) When containing the alt attribute describes to search engines the content of the image.  
D.) Both A and C.

12. Describe the purpose of the &lt;a&gt; element.  
A.) Sets the browser to alarm an alert sound.  
B.) Defines an anchor link for the purpose of navigating between pages.  
C.) Defines an automatic closing of the page.  
D.) None of the above.

13. It is the best practice to group website files (and subfolders) into a single folder.  
A.) TRUE  
B.) FALSE

14. It is impossible to validate HTML to check if it is written correctly.  
A.) TRUE  
B.) FALSE

15. The &lt;th&gt; element defines a heading cell within a table.  
A.) TRUE  
B.) FALSE

16. Fill in the blank: The file name of the home page (first page you want to load) in HTML must be called _________.  
A.) index.html  
B.) homepage.htm  
C.) anything.html  
D.) None of the above.

17. Which of the following code demonstrates an HTML comment?  
A.) // This is a comment in HTML  
B.) /* This is a comment in HTML */

## Answer Sheet

1. **B**, The &lt;!doctype&gt; element informs the browser the mark-up language being used and its version. It should always appear as the first element at the top of the document to insure proper display and validation. Once a version is declared you should only use syntax accepted by that version.  
See video: [http://www.youtube.com/watch?v=RBQX-Ko7A_s](http://www.youtube.com/watch?v=RBQX-Ko7A_s)

2. **C**, The &lt;html&gt; element defines the starting and ending points of the html code in the document and also accepts an attribute of lang that suits the purpose of specifying the main language of the content in the page.  
See video: [http://www.youtube.com/watch?v=RBQX-Ko7A_s](http://www.youtube.com/watch?v=RBQX-Ko7A_s)

3. **A**, The &lt;head&gt; element defines the section of the HTML document that contains the title, meta data, and other browser related script, css, and shortcut icon links. This section is meant for machines to read and is not visible to our eye when browsing the page, unlike the body that contains physical visible content.  
See video: [http://www.youtube.com/watch?v=RBQX-Ko7A_s](http://www.youtube.com/watch?v=RBQX-Ko7A_s)

4. **D**, The &lt;body&gt; element defines the section of the HTML document that contains all of the visible page content such as images, text, links, and page layout.  
See video: [http://www.youtube.com/watch?v=RBQX-Ko7A_s](http://www.youtube.com/watch?v=RBQX-Ko7A_s)

5. **D**, The &lt;title&gt; element defines the pages main title. This appears at the top of the browser tab. It is important to search engines.  
See video: [http://www.youtube.com/watch?v=RBQX-Ko7A_s](http://www.youtube.com/watch?v=RBQX-Ko7A_s)

6. **D**, The &lt;h1&gt; element defines a level 1 heading which make sthe text larger by default. It is the largest text heading and also carries the most importance for search engines amongst the other heading elements.  
See video: [http://www.youtube.com/watch?v=toswcv5oj9I](http://www.youtube.com/watch?v=toswcv5oj9I)

7. **D**, The &lt;p&gt; element defines a paragraph of text. It implies readable content, and allows spacing for separation of blocks of text.  
See video: [http://www.youtube.com/watch?v=toswcv5oj9I](http://www.youtube.com/watch?v=toswcv5oj9I)

8. **A**, The &lt;ul&gt; element defines an unordered list of items. Each piece of content inside the list is then surrounded by &lt;li&gt; elements to separate each item in the list.  
See video: [http://www.youtube.com/watch?v=2IIhsgcNU-M](http://www.youtube.com/watch?v=2IIhsgcNU-M)

9. **C**, The &lt;ol&gt; element defines an ordered list of items. Each piece of content inside the list is then surrounded by &lt;li&gt; elements to separate each item in the list.  
See video: [http://www.youtube.com/watch?v=2IIhsgcNU-M](http://www.youtube.com/watch?v=2IIhsgcNU-M)

10. **B**, The &lt;li&gt; element defines an individual list item to exist inside either &lt;ol&gt; or &lt;ul&gt; parent elements.  
See video: [http://www.youtube.com/watch?v=2IIhsgcNU-M](http://www.youtube.com/watch?v=2IIhsgcNU-M)

11. **D**, The &lt;img&gt; element defines an image to be loaded inside the document. The images src attribute points to the location of the image and the alt attribute tells search engines about the content of the image, and also is read aloud using screen readers for the visually impaired.  
See video: [http://www.youtube.com/watch?v=VQjS7umZeGc](http://www.youtube.com/watch?v=VQjS7umZeGc)

12. **B**, The &lt;a&gt; element defines an anchor link for the purpose of navigating between pages. The &lt;a&gt; element accepts the title attribute that allows a caption to appear momentarily on mouse hover, as well as the target attribute that accepts values such as _blank for the purpose of opening a new browser tab upon navigating.  
See video: [http://www.youtube.com/watch?v=_vZ7x_nFSl0](http://www.youtube.com/watch?v=_vZ7x_nFSl0)

13. **A**, True, it is a best practice to group files and subfolders for a web project into a single folder that way their link relationships are maintained locally, as well as when the files are moved to the server.  
See video: [http://www.youtube.com/watch?v=TmW9d8Uik6E](http://www.youtube.com/watch?v=TmW9d8Uik6E)

14. **B**, False, you may validate your code online or using browser plugins to check if your code is written correctly.  
See video: [https://www.youtube.com/watch?v=nYglnxMUixM](https://www.youtube.com/watch?v=nYglnxMUixM)

15. **A**, True, The &lt;th&gt; element formats the cell inside tables to be a heading cell which by default has bold text and is centered inside the cell to help label that column inside the table.  
See video: [http://www.youtube.com/watch?v=mHeCqIctSRU](http://www.youtube.com/watch?v=mHeCqIctSRU)

16. **A**, index.html or index.htm are the only accepted file names for html files to load automatically when a browser makes a request to a remote domain the first time.  
See video: [http://www.youtube.com/watch?v=i61lTJ6OpDE](http://www.youtube.com/watch?v=i61lTJ6OpDE)

17. **C**, &lt;!-- This is an html comment, they can be single or multi line. They create sections of code the browser will ignore, so we can disable sections or leave ourselves notes in the code --&gt;  
See video: [http://www.youtube.com/watch?v=RBQX-Ko7A_s](http://www.youtube.com/watch?v=RBQX-Ko7A_s)

18. **D**, This example started out defining a &lt;ul&gt; unordered list (line 02), however, on line 06 the unordered list is improperly terminated with the &lt;/ol&gt; which indicates the close of for an ordered list (the wrong list type). The correct code on line 06 should read &lt;/ul&gt; instead.  

19. **A**, In this example line 01 has a missing &gt; symbol after h2 in the opening &lt;h2&gt; tag.
